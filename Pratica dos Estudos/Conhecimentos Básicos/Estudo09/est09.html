Neste código estarei explorando mais tipos de formatação para textos em páginas web.

Códigos-fonte

Em python, a sintaxe do comando print é bem simples, a exemplo de: 

    peso = 70
    altura = 1.62
    idade = 22


    imc = peso / (altura ** 2)

    print('\nDuas formas de formatar um texto !')

    print("O peso é: {}, a altura é {} e a idade é: {}.\nUtilizando esses dados, o IMC fica igual a: {:.1f} !" .format(peso, altura, idade, imc))

    print('\n', imc)
